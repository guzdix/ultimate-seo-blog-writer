# User Input Format

The user should provide:

- Website URL
- Brand description
- Primary keyword
- Content angle
- Word count target
- CTA goal
- Optional personal experience

Example:

Website: https://example.com
Brand: Luxury villa rental in Bali
Keyword: bali villa investment
Angle: mistakes foreigners make
Word Count: 2000
CTA: contact consultation
Experience: helped 12 investors buy villas in Bali
